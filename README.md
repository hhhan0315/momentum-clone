## 참고사항
- let, const, var
  1. var로 변수를 한번 더 선언해도 오류가 발생하지 않는다. ES6 이후로 let, const
  2. let은 변수에 재할당이 가능, const는 불가능
  3. 기본적으로 const 선언하고 재할당이 필요한 경우에 let을 선언.
- Data Type
  - array : const data = ["value"];
  - object : const data = {key:"value",};
